✔ What is base64?

  • Base64 : String → ASCII Binary → 6Bit Cut → Encode
  • 6Bit Cut : 6비트씩 묶어서 임의의 64개 데이터 값 참조후 치환
  • Padding : 6비트씩 묶고 남는 비트 이퀄싸인 (=)
  • 64개 데이터 : Standard (RFC 4648) → A-Za-z0-9+/=
